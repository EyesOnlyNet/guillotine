[[GUILLOTINE]]

PLAYFLOW
--------

1) select player 
    --> input names
2) start game
    --> init game
        --> set day to 0
        --> set playerCards to 0
    --> mix actionCardStack
    --> for each player:
        --> get actionCards (default: 5)
    --> mix nobleCardStack
    --> build beheadQueue 
        --> draw from nobleCardStack (default: 12 cards/times)
        --> setDirection (default: ltr)
    --> place guillotine (on head of beheadQueue)
    --> select start-player (default: user-selection || random)
3) play
    --> while days not over (default: 3) and beheadQueue not empty
        --> show statistic
        --> show stacks
        --> while beheadQueue not empty
            --> show beheadQueue
            --> show playerCards
            --> play turn
                --> play action card (default: 1)
                    --> play
                    --> postPlay
                --> behead
                    --> preBehead
                    --> behead
                    --> postBehead
                --> draw action card (default: 1)
                    --> draw
                    --> postDraw
        --> start next day
            --> refill beheadQueue (default: 12)
4) play actionCards
    --> only pointCards
5) calculate winner
